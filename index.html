Data Source

The data used to display the graphs on this site is from the JSON time-series from pomber on GitHub. The JSON time-series project takes the data from CSSEGISandData/COVID-19 and merges it into a JSON file.

CSSEGISandData/COVID-19 is a GitHub-repository operated by the Johns Hopkins University Center for Systems Science and Engineering and is used to supply their Coronavirus-Dashboard. You can read more about it on their README page on GitHub.
